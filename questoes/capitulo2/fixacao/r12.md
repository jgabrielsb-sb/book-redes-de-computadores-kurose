Enunciado: Considere um site de comércio eletrônico que quer manter um registro de compras para cada um dos seus clientes. Descreva como isso pode ser feito com cookies.

The browser, when requesting a service, can be linked by a certain integer number or id by the website server. That id will be associated to a certain browser and everytime that it acess the website the informations that are linked to that id will be associated to it. That funcionalitie can be provided by cookies.
